Kiểu dữ liệu đơn giản (primitive type ) kiểu tham trị 1. String VD: "Lan", 'Mai',... 2. number VD: 10,2,3,2,.. 3. null VD: null 4. undefined VD: undefined 5. NaN VD: NaN 6. Boolean VD: true / false Kiểu dữ liệu phức tạp (reference type) kiểu tham chiếu 1. array (mảng) VD: [1,2,3,4,5] 2. object VD: {name:"Lan", gender:"nữ", age:18} 3. function VD: function render(){ //thực thi }
